# jobs/

Scheduled / batch jobs (cron-style tasks that run outside the request path). **None yet.**

When a job is added, it lives here as a self-contained script or workspace and is scheduled via
n-easyapp's deploy-cronjob cap for project `stemrobin`. Kept as a placeholder so the unified
prodfarm layout (`app/` · `infra/` · `jobs/` · `ssot-schemas/`) is complete and discoverable.
