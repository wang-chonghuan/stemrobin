数学 · 第 2 阶段

整式加减

把两个整式整体相加或相减，化到最简——做法就是把每个整式括起来，再去括号、合并同类项。一句话：整式加减 = 去括号 + 合并同类项。
整式加减 去括号 + 合并同类项 前置：去括号 / 合并同类项
1为什么学这个

这一阶段一路走来，我们已经把带字母的式子会读、会写、会求值，最近两课还练熟了两件事：合并同类项（把并排的同类项收拢，系数相加、字母不变）和去括号（把括号拆掉，减号让括号里各项变号）。两件武器都磨好了。

可真正的题目往往不是给你一个孤零零的式子，而是给你两个整式，让它们整体相加或相减。比如：

$$(3a + 2) + (a - 5) \qquad (2x - 1) - (x - 3)$$

第一个是两个式子相加，第二个是两个式子相减。它们各自都不复杂，可一旦要"加起来"或"减掉"，问题就来了：$3a+2$ 和 $a-5$ 怎么合到一起？尤其相减的那个——$(2x-1)$ 要减去整个 $(x-3)$，是只减 $x$ 呢，还是连 $-3$ 也一起处理？符号一不小心就错。

本节的驱动问题： 两个整式相加或相减，结果要怎么一步步算出来、化到最简？特别是相减的时候，被减的那个括号里各项符号该怎么变？

这件事叫整式加减，它是第二阶段的收官——把前两课的"去括号"和"合并同类项"接成一条完整的流水线。带着上面"相减时符号怎么处理"的问题往下读。

2讲解

先把"要加减的东西"说清楚。我们加减的对象叫整式——它是单项式与多项式的统称（像 $3a$、$5$ 这样的单项式，和像 $x^2-1$ 这样的多项式，都算整式）。判断的关键：分母里不能含字母。分母含字母的（如 $\dfrac1x$）是分式，不是整式。

$3a + 2$（多项式） $x^2 - 1$（多项式） $5$（单项式，常数） $\dfrac{1}{x}$（分母含字母，是分式） $\dfrac{a+1}{b}$（分母含字母 $b$，不是整式）

整式加减的做法：括起来 → 去括号 → 合并同类项

整式加减：把参与加减的每个整式用括号括起来——加就写成 $+(\;)$，减就写成 $-(\;)$——然后去括号（这一步用第 7 课：减号让括号里每一项变号），最后合并同类项（这一步用第 6 课）。一句话记住：

整式加减 = 去括号 + 合并同类项。

$(2x - 1) - (x - 3) = 2x - 1 - x + 3 = x + 2$（减号让 $x$、$-3$ 都变号） $(2x - 1) - (x - 3) = 2x - 1 - x - 3$（末项 $-3$ 漏变号，应是 $+3$）

为什么相减要"每一项都变号"

相减时，被减的整式前面顶着一个减号，比如 $-(x-3)$。这个减号就是因子 $-1$：“$-(\;)$” 等于 “$\times(-1)$”。由分配律，$-1$ 要乘进括号里每一项——$x$ 变 $-x$，$-3$ 变 $+3$。所以减一个整式，等于把它每一项都变成相反数再加上去。这正是第 7 课"减号后各项变号"的规则，一个都不能漏。

它把前两课直接接起来。 去括号（第 7 课）解决"括号和符号"，合并同类项（第 6 课）解决"把同类凑到一起"；两者共同的引擎都是分配律。整式加减就是这两步的组合：先去括号打开门，再合并同类项收拾屋子。学会它，你就真正会"摆弄"带字母的式子了——这是第二阶段的收官动作。
三个最常踩的坑：
（1）相减时只变第一项、漏变后面项。 $(2x-1)-(x-3)$ 常被写成 $2x-1-x-3$——末项 $-3$ 忘了翻成 $+3$。正确是 $2x-1-x+3$：被减括号里每一项都要变号，一个都不能漏。
（2）带系数时漏乘后面的项。 $2(a-1)$ 常被写成 $2a-1$（只乘了第一项）。正确是 $2a-2$：系数要乘每一项。
（3）合并时改动了字母部分或指数。 合并同类项只把系数相加，字母和指数都不变：$2x^2$ 与 $x^2$ 合并是 $3x^2$，指数还是 $2$，不会变成 $x^4$。

把它画出来

看最容易错的相减 $(2x-1)-(x-3)$：整个过程就是"去括号 → 合并同类项"两步，减号让 $x$、$-3$ 变号。

3例题
例 1　计算 $(3a + 2) + (a - 5)$。
1.两个整式相加。前面是加号，去括号后各项符号不变：$3a + 2 + a - 5$。
2.合并同类项：$3a$ 与 $a$ 合并得 $4a$；常数 $2$ 与 $-5$ 合并得 $-3$。
3.结果 $4a - 3$。
例 2　计算 $(2x - 1) - (x - 3)$。（相减，重点看变号）
1.后一个括号前是减号，等于因子 $-1$，括号里每一项都变号：$x$ 变 $-x$，$-3$ 变 $+3$。得 $2x - 1 - x + 3$。
2.合并同类项：$2x - x = x$；$-1 + 3 = +2$。
3.结果 $x + 2$。（末项 $-3$ 一定要翻成 $+3$，不能漏。）
例 3　计算 $(2x^2 - x + 1) - (x^2 + 3x - 4)$。（二次多项式相减）
1.后括号前是减号，里面三项全变号：$x^2$ 变 $-x^2$，$3x$ 变 $-3x$，$-4$ 变 $+4$。得 $2x^2 - x + 1 - x^2 - 3x + 4$。
2.按同类项分组合并：$2x^2 - x^2 = x^2$；$-x - 3x = -4x$；$1 + 4 = 5$。
3.结果 $x^2 - 4x + 5$。（注意 $x^2$ 的指数始终是 $2$，合并只动系数。）
例 4　化简 $2(a - 1) + 3(a + 2)$。（带系数）
1.先用分配律去括号，系数乘进每一项：$2(a-1) = 2a - 2$；$3(a+2) = 3a + 6$。得 $2a - 2 + 3a + 6$。
2.合并同类项：$2a + 3a = 5a$；$-2 + 6 = +4$。
3.结果 $5a + 4$。
例 5　说理：为什么"减一个整式"要把它每一项都变号？
1.减去整式 $A-B$，写出来是 $-(A - B)$，括号前的减号就是因子 $-1$：$-(A-B) = (-1)\times(A - B)$。
2.由分配律，$-1$ 要乘进括号里每一项：$(-1)\times A = -A$，$(-1)\times(-B) = +B$。
3.所以每一项都翻号，$-(A-B) = -A + B$。关键就是"$-1$ 乘到每一项"——不是只翻第一项。这也说明：整式加减靠的正是去括号 + 合并同类项，底下是分配律。
4与其他知识点的联系
  • 往回看（这节课站在什么上面）：加减的对象是整式，也就是单项式与多项式的统称——这些概念本阶段前面都学过。做法直接调用两件工具：去括号（第 7 课，负责括号与符号）和合并同类项（第 6 课，负责把同类凑到一起）。而"减号让每一项变号"背后是相反数与分配律。可以说，整式加减没有新招式，全是把学过的动作串起来用。
  • 往前看（这节课通向哪里）：紧接着第三阶段的解方程——方程两边都是整式，移项、合并同类项整理之后才能解出未知数，而整理用的正是整式加减。再往后学整式乘法、分式时，化简每一步也离不开去括号与合并同类项。整式加减是往后所有代数运算的地基动作。
  • 贯穿的抽象主线：分配律 → 去括号 + 合并同类项 → 整式加减（化简）。分配律是引擎：正着用它把括号拆开（去括号），逆着用它把同类项收拢（合并同类项），两步合起来就是整式加减。这也是整个第二阶段的收官——学完这一阶段，你就能把带字母的式子读、写、求值、化简全套走通。第 1 节问的是"两个整式怎么加减到最简"（制造需求）；这里讲的是"它接住了哪些旧知识、通向哪些新知识"（搭起结构）。
5概念口试

下面几个问题不用算，只用一句话答出来，检查你是不是真的"懂"了。先自己答；答不上来，就回到上面「讲解」里找——这些概念课文里都讲过。

  1. 什么是整式？用一句话说清楚。
    整式是单项式与多项式的统称；判断关键是分母里不含字母（分母含字母的是分式，不是整式）。
  2. 整式加减分成哪两步？
    先去括号（相减时括号里每一项变号），再合并同类项。一句话：整式加减 = 去括号 + 合并同类项。
  3. 两个整式相减时，被减的那个括号里各项符号会怎么变？
    每一项都要变号（变成相反数），一个都不能漏——因为括号前的减号相当于因子 $-1$，乘进每一项都翻号。
  4. $\dfrac{1}{x}$ 是整式吗？为什么？
    不是。它的分母含字母 $x$，属于分式；整式要求分母不含字母。
  5. 整式加减靠的是哪条道理？
    分配律。去括号是正着用分配律把因子乘进每一项，合并同类项是逆着用分配律把同类项收拢，两步都由分配律支撑。
  6. 合并同类项时，字母部分和指数会变吗？
    不变。合并只把同类项的系数相加，字母和指数保持不变（如 $2x^2 + x^2 = 3x^2$，指数仍是 $2$）。
6练习
  1. 辨认下面哪些是整式：$3a+2$、$\dfrac{1}{x}$、$x^2-1$、$\dfrac{a+1}{b}$、$5$？
    整式：$3a+2$、$x^2-1$、$5$。不是整式：$\dfrac{1}{x}$、$\dfrac{a+1}{b}$（分母含字母，是分式）。
  2. 辨认这一步整式相减对不对：$(2x-1)-(x-3) = 2x-1-x+3$？
    对。后括号前是减号，里面每一项变号：$x$ 变 $-x$，$-3$ 变 $+3$，所以是 $2x-1-x+3$（再合并得 $x+2$）。
  3. 表示把 $-(x - 3)$ 写成去括号后不带括号的形式。
    $-(x-3) = -x + 3$。减号是因子 $-1$，两项都翻号：$x$ 变 $-x$，$-3$ 变 $+3$。
  4. 表示把 $(2x-1)-(x-3)$ 写成去括号后（还没合并）的形式。
    $(2x-1)-(x-3) = 2x - 1 - x + 3$。前括号加号各项不变；后括号减号让 $x$、$-3$ 变号。
  5. 操作计算 $(3a + 2) + (a - 5)$。
    去括号（加号，不变）：$3a + 2 + a - 5$。合并：$3a+a=4a$，$2-5=-3$。结果 $4a - 3$。
  6. 操作计算 $(2x - 1) - (x - 3)$。
    去括号（减号变号）：$2x - 1 - x + 3$。合并：$2x-x=x$，$-1+3=+2$。结果 $x + 2$。
  7. 操作化简 $2(a - 1) + 3(a + 2)$。
    去括号：$2a - 2 + 3a + 6$。合并：$2a+3a=5a$，$-2+6=+4$。结果 $5a + 4$。
  8. 操作计算 $(2x^2 - x + 1) - (x^2 + 3x - 4)$。
    去括号（后括号三项全变号）：$2x^2 - x + 1 - x^2 - 3x + 4$。合并：$2x^2-x^2=x^2$，$-x-3x=-4x$，$1+4=5$。结果 $x^2 - 4x + 5$。
  9. 操作化简 $(4m + 3) - (m - 2) - 1$。
    去括号：$4m + 3 - m + 2 - 1$。合并：$4m-m=3m$，$3+2-1=4$。结果 $3m + 4$。
  10. 反推已知两个整式的和是 $5a - 3$，其中一个是 $3a + 2$，另一个是什么？
    另一个 $=(5a-3)-(3a+2)=5a-3-3a-2=2a-5$。（用差反推：从和里减去已知那个。）
  11. 反推已知 $(kx + 1) + (2x - 3) = 5x - 2$，求 $k$。
    左边合并：$(k+2)x + (1-3) = (k+2)x - 2$。与 $5x-2$ 比较，$k+2=5$，所以 $k=3$。
  12. 反推某整式减去 $(x - 4)$ 后得到 $3x + 1$，这个整式是什么？
    设它为 $A$，则 $A-(x-4)=3x+1$，所以 $A=(3x+1)+(x-4)=3x+1+x-4=4x-3$。
  13. 辨错有人写 $(2x - 1) - (x - 3) = 2x - 1 - x - 3$。错在哪？正确结果是什么？
    错在末项 $-3$ 漏变号。后括号前是减号，每一项都翻号，$-3$ 应变 $+3$。正确：$2x-1-x+3 = x + 2$。
  14. 辨错有人写 $2(a - 1) = 2a - 1$。错在哪？
    系数 $2$ 漏乘第二项。分配律要乘每一项：$2\times(-1)=-2$。正确 $2a - 2$。
  15. 辨错有人写 $2x^2 + x^2 = 3x^4$。错在哪？
    合并同类项只把系数相加、指数不变：$2x^2 + x^2 = 3x^2$，指数仍是 $2$，不能相加变成 $x^4$。
