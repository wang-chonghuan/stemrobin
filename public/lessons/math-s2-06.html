数学 · 第 2 阶段

合并同类项

把同类项的系数加到一起、字母部分不变——让一个长式子变短，而它代表的数始终不变。
合并同类项 分配律 前置：同类项
1为什么学这个

上一节课，我们学会了在一串项里看出"谁和谁是同一类"——只比字母指纹，不看系数。但我们当时故意停住了：找出了同类项，却没有把它们合到一起。

现在来看一个真实的式子：

$$6m^2 + 2mn - m^2 + 5 - 4mn - 1$$

它一共有六项，又长又笨。可你只要瞄一眼就会发现：$6m^2$ 和 $-m^2$ 是一类，$2mn$ 和 $-4mn$ 是一类，$5$ 和 $-1$ 又是一类。既然每一类里的项"长得几乎一样"，把它们各自凑到一起，式子是不是就能写得短很多？

但这里藏着一个必须先回答的疑问：式子代表的是一个数（给 $m$、$n$ 各代一个具体值它就有一个值）。如果我随手把它改短，会不会一不小心就把这个数也改掉了？短了却算错了，那就没意义。

本节的驱动问题： 能不能把这样一个长式子改写得更短，而它所代表的数始终不变？

这一步叫化简。它现在才出现得正是时候：我们已经会用字母写式子、会判同类项，就差"把同类项真正合起来"这最后一步。带着上面这个问题往下读。

2讲解

规则：只加系数，字母部分原样不动

先把标准说法摆出来，再讲清它为什么成立。

合并同类项：把同类项的系数相加，字母部分保持不变，写成一个项。不是同类项的，不能合并，原样保留。

比如 $3x^2$ 和 $5x^2$ 是同类项（指纹都是 $x^2$），合并就是把系数 $3$ 和 $5$ 加起来：$3x^2 + 5x^2 = 8x^2$。注意结果是 $8x^2$，字母部分还是 $x^2$，一点没变。

为什么能这样合？——靠分配律

这条规则不是规定出来的，它有理由，理由就是分配律。

分配律说：$ax + bx = (a+b)x$。也就是说，如果两项有一个公因子 $x$，就可以把 $x$ 提出来，让两个系数 $a$、$b$ 先加起来。

同类项恰好满足这个条件：它们的字母部分完全相同——那相同的字母部分，正是那个可以提出来的公因子。于是：

$$3x^2 + 5x^2 = (3+5)\,x^2 = 8x^2$$

看清这里谁变了、谁没变：

  • 不变的是字母部分 $x^2$（连同它的指数 $2$）——它是公因子，被原封不动地提出来。
  • 变的只有系数：$3$ 和 $5$ 相加成 $8$。

这正好接上一课那句话：字母指纹是不变量，系数才是可变量。上一课我们用"字母部分相同"把项分了类；这一课就靠"字母部分相同"把它当公因子提出去，把系数加起来。

为什么合并以后代表的数不变？ 因为 $3x^2 + 5x^2 = (3+5)x^2$ 只是分配律的逆用，是一个恒等式——不管 $x$ 取什么数，等号两边都相等。验一下：令 $x=2$，左边 $3\times4 + 5\times4 = 12+20 = 32$，右边 $8\times4 = 32$，确实一样。所以"变短"和"值不变"可以同时做到，这就回答了第 1 节的问题。

记号与读法

合并写出来常带一个中间步骤：$7a - 2a = (7-2)a = 5a$。中间的 $(7-2)a$ 把"系数在相加、字母 $a$ 被提出来"显式地写了出来，熟练后可以直接写 $5a$。这里的减号要连着系数一起算：$-2a$ 的系数是 $-2$，所以是 $7 + (-2) = 5$。

三个最常踩的坑：
（1）把不是同类项的也合了。 $3x + 2y$ 不能合成 $5xy$——$x$ 和 $y$ 不是同一个字母，没有公因子，分配律用不上，只能原样保留。$x^2 + x^3$ 也不能合，指数不同就不是同类项。
（2）合并时连指数也加了。 $3x^2 + 5x^2 = 8x^2$，不是 $8x^4$。相加的只有系数 $3+5=8$；字母部分和指数都是被提出来的公因子，原样不动。
（3）漏看符号。 系数前的正负号要带着算：$-4mn + 2mn = (-4+2)mn = -2mn$，不是 $-6mn$、也不是 $2mn$。

把它画出来

合并就像把同样的方块数到一堆：系数相加，方块上的标签（字母部分）不变；标签不同的方块凑不到一起。

3例题
例 1　合并 $3x^2 + 5x^2$。
1.先确认是不是同类项：两项字母部分都是 $x^2$，指纹相同，可以合并。
2.把相同的字母部分 $x^2$ 当公因子提出来（这就是分配律）：$3x^2 + 5x^2 = (3+5)x^2$。
3.只把系数加起来：$(3+5)x^2 = 8x^2$。字母部分仍是 $x^2$，结果是 $8x^2$。
例 2　合并 $-4mn + 2mn$。
1.两项指纹都是 $mn$，是同类项。系数分别是 $-4$ 和 $2$（负号算进系数）。
2.提出公因子 $mn$，系数相加：$-4mn + 2mn = (-4+2)mn$。
3.$-4+2 = -2$，所以结果是 $-2mn$。注意符号别丢。
例 3　化简 $6m^2 + 2mn - m^2 + 5 - 4mn - 1$（即第 1 节那个长式子）。
1.先按指纹分组（这一步是上一课学的）：
　指纹 $m^2$：$6m^2$、$-m^2$；
　指纹 $mn$：$2mn$、$-4mn$；
　常数项：$5$、$-1$。
2.每组各自合并，系数相加、字母部分不变：
　$6m^2 - m^2 = (6-1)m^2 = 5m^2$；
　$2mn - 4mn = (2-4)mn = -2mn$；
　$5 - 1 = 4$。
3.把三组结果连起来（不同类的之间不能再合，原样保留）：$5m^2 - 2mn + 4$。
4.六项缩成了三项，又短又整齐。这就是"分类 → 合并"的接力：先分箱，再把每箱的系数加起来。
例 4　化简 $7a + 3b - 2a$，并解释：哪两项能合、为什么；$3b$ 为什么不能动。
1.分组：$7a$ 和 $-2a$ 指纹都是 $a$，是同类项；$3b$ 指纹是 $b$，自己一组。
2.$7a$ 和 $-2a$ 能合，因为它们有公因子 $a$，分配律给出 $7a - 2a = (7-2)a = 5a$。
3.$3b$ 不能和它们合，因为它的字母是 $b$，和 $a$ 不是同一个字母，没有公因子，分配律用不上，只能原样留着。
4.结果：$5a + 3b$。它已经最简——再没有两项是同类项了。
4与其他知识点的联系
  • 往回看（这节课站在什么上面）：它直接接着上一课的同类项——必须先判出"谁和谁同类"，才谈得上合并；还用到用字母表示数（式子代表的是一个数，所以才要求合并后值不变）；而合并合法的依据是更早的分配律 $ax+bx=(a+b)x$，公因子就是那相同的字母部分。
  • 往前看（这节课通向哪里）：下一课去括号会先把括号拆开，拆完往往冒出一堆同类项，要靠合并收尾；再下一课整式加减，本质就是"把两个式子的项放在一起、反复合并同类项"。再往后到第三阶段解方程，移项之后两边也都要合并同类项，才能解出未知数。合并同类项是这一串运算反复用到的基本动作。
  • 贯穿的抽象主线：分类 → 合并（系数相加、字母不变）→ 化简。上一课把项按字母指纹分了类，分清了不变量（字母部分）和可变量（系数）；这一课就利用"同类项有公因子"，用分配律把每一类的系数加起来，让式子变短而值不变。这条主线会一路撑起整个整式运算。
5练习
  1. 辨认$5x^2$ 和 $-3x^2$ 这两项能不能合并？$5x^2$ 和 $5x^3$ 呢？
    答案
    $5x^2$ 和 $-3x^2$ 能合并（指纹都是 $x^2$，是同类项）。$5x^2$ 和 $5x^3$ 不能（指数一个是 $2$、一个是 $3$，不是同类项）。
  2. 辨认$4ab$ 和 $4ba$ 能不能合并？为什么？
    答案
    能。$ab = ba$，两项指纹相同、是同类项，公因子 $ab$ 可以提出来：$4ab + 4ba = (4+4)ab = 8ab$。
  3. 表示把 $9y - 2y$ 的合并过程用 $(a+b)x$ 的形式写出来，再得最简结果。
    答案
    $9y - 2y = (9-2)y = 7y$。先提出公因子 $y$，再把系数 $9$ 与 $-2$ 相加。
  4. 表示用一句话说明：合并 $8c^2 + c^2$ 时，"加"的是什么、"不变"的是什么，并写出结果。
    答案
    相加的是系数 $8$ 和 $1$（$c^2$ 的系数是 $1$），不变的是字母部分 $c^2$。结果 $8c^2 + c^2 = (8+1)c^2 = 9c^2$。
  5. 操作计算 $12mn - 5mn$。
    答案
    $12mn - 5mn = (12-5)mn = 7mn$。
  6. 操作计算 $-6a + 10a - a$。
    答案
    三项都是指纹 $a$ 的同类项：$(-6+10-1)a = 3a$。
  7. 操作化简 $4x^2 + 3x - x^2 + 2x$。
    答案
    分两组：$x^2$ 组 $4x^2 - x^2 = 3x^2$；$x$ 组 $3x + 2x = 5x$。结果 $3x^2 + 5x$（两项不同类，不能再合）。
  8. 操作化简 $7p + 2q - 3p + 6 - 5q - 1$。
    答案
    $p$ 组 $7p-3p=4p$；$q$ 组 $2q-5q=-3q$；常数项 $6-1=5$。结果 $4p - 3q + 5$。
  9. 反推某两个同类项合并后得 $5x^2$，其中一项是 $8x^2$，另一项是什么？
    答案
    设另一项系数为 $b$，则 $8 + b = 5$，得 $b = -3$，另一项是 $-3x^2$。
  10. 反推$3a^2 + k a^2 = 10a^2$，求 $k$。
    答案
    系数相加：$3 + k = 10$，所以 $k = 7$。
  11. 反推已知 $7mn - \square = 2mn$，方框里应填哪个项？
    答案
    设填 $c\cdot mn$，则 $7 - c = 2$，$c = 5$，方框填 $5mn$。
  12. 辨错有人写 $2x^2 + 3x^2 = 5x^4$。错在哪？正确结果是什么？
    答案
    错在把指数也加了。合并只把系数相加（$2+3=5$），字母部分 $x^2$ 是公因子、原样不动。正确结果 $5x^2$。
  13. 辨错有人写 $3x + 2y = 5xy$。错在哪？
    答案
    $3x$ 和 $2y$ 不是同类项（字母一个是 $x$、一个是 $y$），没有公因子，分配律用不上，不能合并。$3x + 2y$ 已经最简，原样保留。
  14. 辨错有人写 $-4mn + 2mn = -6mn$。错在哪？
    答案
    漏看了符号。系数相加是 $-4 + 2 = -2$，不是 $-6$。正确结果 $-2mn$。
