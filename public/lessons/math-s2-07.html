数学 · 第 2 阶段

去括号

把式子里的括号拆掉——本质是用分配律把括号前的那个因子（+1、−1 或一个系数）分配到括号内每一项，符号跟着变。
去括号 分配律 前置：合并同类项
1为什么学这个

上一节课我们学会了合并同类项：把同类的项凑到一起，系数相加、字母不变，让式子变短而值不变。可它有个前提——同类项得并排摆着，你才好把它们收拢。

但真实的式子常常带着括号，同类项被括号关在里面，凑不到一块儿。比如：

$$3a + 2(a-1) \qquad 5x - (2x-3)$$

第一个式子里，$3a$ 在括号外，$a$ 却锁在括号内；第二个式子里 $5x$ 和括号里的 $2x$ 明明是同类，可中间隔着一个括号、括号前还顶着一个减号。括号不拆开，合并同类项这一步就无从下手。

那就把括号去掉呗。但麻烦正在这里：括号前是加号、是减号、还是一个数，去掉之后括号里的符号会不会变？变哪些？$5x-(2x-3)$ 去了括号，里面的 $-3$ 到底该变成 $+3$ 还是 $-3$？猜错一个符号，整道题就错了。

本节的驱动问题： 括号到底怎么去掉？尤其是括号前面是减号、或者括号前带着一个数的时候，括号里各项的符号会怎么变？

这一步叫去括号。学会它，才能把藏在括号里的同类项放出来、接着合并。带着上面这个"符号怎么变"的问题往下读。

2讲解

去括号要分三种情形看：括号前是加号、是减号、还是有一个数（系数）。三种情形规则不同，但底下是同一条道理——分配律。我们先一种一种把规则说清，最后再点破这条共同的道理。

情形一：括号前是加号——各项符号不变

去括号：把括号连同它前面的符号一起去掉。当括号前是加号（“$+$”）时，去掉后括号里各项的符号都不变：

$$a + (b - c) = a + b - c$$

$x + (y - 2) = x + y - 2$ $3a + (a - 1) = 3a + a - 1$ $x + (y - 2) = x + y + 2$（末项本是 $-2$，符号不该变）

情形二：括号前是减号——各项都变号

当括号前是减号（“$-$”）时，去掉括号和它前面的减号后，括号里每一项都要变号（变成它的相反数）：正变负、负变正。

$$a - (b - c) = a - b + c$$

$-(x - 3) = -x + 3$（$x$ 变 $-x$，$-3$ 变 $+3$） $5x - (2x - 3) = 5x - 2x + 3$ $-(x - 3) = -x - 3$（漏给第二项变号，$-3$ 没翻成 $+3$）

为什么减号这么"厉害"？因为括号前的减号相当于乘以 $-1$：“$-(\;)$” 就等于 “$\times(-1)$”。$-1$ 乘进去，每一项都被乘了一次，所以每一项都翻号。这一点下面还会再点。

情形三：括号前有一个数——用分配律乘进每一项

当括号前是一个数（系数）时，就是标准的分配律：把这个数乘到括号里的每一项上，连同各项的符号一起乘：

$$2(x + 3) = 2x + 6 \qquad -3(a - 2) = -3a + 6$$

$2(x + 3) = 2x + 6$ $-3(a - 2) = -3a + 6$（$-3 \times a = -3a$，$-3 \times (-2) = +6$） $-3(a - 2) = -3a - 6$（第二项漏乘符号：$-3\times(-2)$ 应得 $+6$）

特别当系数是负数时（如 $-3$），它把每一项都乘负，往往连符号一起翻——本质上和"情形二"是一回事，只是那里的因子恰好是 $-1$。

三种情形其实是一条道理——分配律。 去括号本质就是把括号前的那个因子分配到括号内每一项：加号前的因子是 $+1$，$+1$ 乘谁都不变，所以各项符号不变；减号前的因子是 $-1$，$-1$ 乘每一项都翻号；数字前的因子就是那个数。所以三条规则只是同一个分配律，因子不同而已。把 “$-(\;)$” 看成 “$\times(-1)$”，你就再也不会记错减号后面该不该变号。这正好接上一课合并同类项用的那条分配律。

去括号之后，通常紧接着合并同类项

去括号很少是终点。拆开括号后，原来隔着括号的同类项就并排到了一起，下一步几乎总是合并同类项。例如：

$$3a + 2(a-1) = 3a + 2a - 2 = 5a - 2$$

先分配律去括号（$2$ 乘进去得 $2a-2$），$3a$ 和 $2a$ 现在挨在一起了，合并成 $5a$，常数 $-2$ 留着。去括号是"打开门"，合并同类项是"收拾屋子"。

三个最常踩的坑：
（1）减号后只变第一项、漏变后面项。 $5x - (2x - 3)$ 常被写成 $5x - 2x - 3$——第二项 $-3$ 忘了翻号。正确是 $5x - 2x + 3$：括号里每一项都要变号，一个都不能漏。
（2）系数漏乘后面的项。 $-3(a - 2)$ 常被写成 $-3a - 2$（只乘了第一项）或 $-3a-6$（第二项符号算错）。正确是 $-3a + 6$：系数要乘每一项，且连同符号 $-3\times(-2)=+6$。
（3）忘了 “$-(\;)=\times(-1)$”。 拿不准减号后面符号怎么变时，把括号前的减号还原成 $\times(-1)$，让 $-1$ 老老实实乘进每一项，符号就不会错。

把它画出来

看最容易错的减号情形 $-(a-b)$：括号前的减号就是因子 $-1$，它要分配给括号里每一项，把每一项都翻号。

3例题
例 1　去括号：$a + (b - c)$。
1.看括号前的符号：是加号。对应"情形一"——去掉后各项符号不变。
2.把括号连同前面的加号去掉，里面 $b$、$-c$ 原样搬出：$a + b - c$。
3.结果 $a + b - c$。（因子是 $+1$，乘谁都不变。）
例 2　去括号：$5x - (2x - 3)$。
1.括号前是减号——对应"情形二"，括号里每一项都要变号。可把这个减号看成 $\times(-1)$。
2.$2x$ 变 $-2x$，$-3$ 变 $+3$（两项都翻号，一个都不漏）：$5x - 2x + 3$。
3.结果 $5x - 2x + 3$。（若接着合并：$5x-2x=3x$，得 $3x+3$。）
例 3　去括号：$-3(a - 2)$。
1.括号前是数 $-3$——"情形三"，用分配律把 $-3$ 乘进每一项，连符号一起乘。
2.第一项：$-3 \times a = -3a$；第二项：$-3 \times (-2) = +6$。
3.结果 $-3a + 6$。（注意第二项号翻成 $+$，因为负数乘负数得正。）
例 4　化简 $3a + 2(a - 1)$（去括号再合并）。
1.先去括号：括号前是数 $2$，分配进每一项 $2 \times a = 2a$，$2 \times (-1) = -2$，得 $3a + 2a - 2$。
2.去括号后 $3a$ 和 $2a$ 并排了，是同类项，合并：$3a + 2a = 5a$。
3.常数 $-2$ 没有同类项，留着。结果 $5a - 2$。这就是"去括号 → 合并同类项"的完整接力。
例 5　解释：为什么 $-(2x - 3) = -2x + 3$？（说理，不只算结果）
1.括号前的减号就是因子 $-1$：$-(2x-3) = (-1)\times(2x-3)$。
2.由分配律，$-1$ 要乘进括号里每一项：$(-1)\times 2x = -2x$，$(-1)\times(-3) = +3$。
3.所以每一项都翻号，得 $-2x + 3$。关键就是"$-1$ 乘到每一项"——不是只翻第一项。
4与其他知识点的联系
  • 往回看（这节课站在什么上面）：去括号的依据是分配律——和上一课合并同类项用的是同一条 $a(b+c)=ab+ac$，只是这里正着用（把括号前的因子乘进去）。它还用到相反数与有理数的符号法则：减号 $=\times(-1)$、负负得正，正是"减号后各项变号"背后的算理。判断去括号后能不能接着合并，又要回到同类项。
  • 往前看（这节课通向哪里）：紧接着的整式加减，本质就是"反复去括号 + 合并同类项"——把几个带括号的式子拆开、再把同类项收拢。再往后到第三阶段解方程，去分母、移项之前也常常要先去括号，才能把方程理成能解的样子。去括号是整式与方程运算里反复用到的基本动作。
  • 贯穿的抽象主线：分配律 → 去括号 → 合并同类项 → 化简。分配律是引擎：正着用它把括号拆开（去括号），逆着用它把同类项收拢（合并同类项），一开一合，把式子化到最简。第 1 节问的是"为什么需要拆括号"（制造需求）；这里讲的是"拆完之后它接到哪、靠什么支撑"（搭起结构）。
5概念口试

下面几个问题不用算，只用一句话答出来，检查你是不是真的"懂"了。先自己答；答不上来，就回到上面「讲解」里找——这些概念课文里都讲过。

  1. 什么叫去括号？用一句话说清楚。
    把式子里的括号连同它前面的符号一起去掉、把括号里的项写出来；本质是用分配律把括号前的因子分配到括号内每一项。
  2. 括号前是加号时，去掉括号后括号里各项的符号会怎样？
    各项符号都不变，原样写出来（因为括号前的因子是 $+1$）。
  3. 括号前是减号时，括号里各项的符号会怎样？
    每一项都要变号（变成相反数），一个都不能漏——因为减号相当于因子 $-1$，乘进每一项都翻号。
  4. 括号前有一个数时，怎么去括号？
    用分配律把这个数乘到括号里的每一项上，连同各项的符号一起乘。
  5. 为什么 “$-(括号)$” 会让括号里每一项都变号？
    因为 “$-(\;)$” 等于 “$\times(-1)$”，由分配律 $-1$ 要乘进括号里每一项，$-1$ 乘任何一项都使它翻号。
  6. 去括号之后，通常紧接着做哪一步？
    合并同类项——去括号把原本隔着括号的同类项放到一起，接着把它们合并、化简。
6练习
  1. 辨认下面去括号对不对：$a + (b - c) = a + b - c$？$x - (y - 1) = x - y - 1$ 呢？
    第一个对（括号前是加号，各项不变）。第二个错：括号前是减号，$-1$ 应变 $+1$，正确是 $x - y + 1$。
  2. 辨认$-(x - 3) = -x + 3$ 这一步去括号对不对？为什么？
    对。括号前是减号（因子 $-1$），每一项都变号：$x$ 变 $-x$，$-3$ 变 $+3$，所以是 $-x + 3$。
  3. 表示把 $-(a - b)$ 写成不带括号的形式。
    $-(a - b) = -a + b$。减号是因子 $-1$，两项都翻号：$a$ 变 $-a$，$-b$ 变 $+b$。
  4. 表示把 $+(m - 2n + 1)$ 写成不带括号的形式。
    $+(m - 2n + 1) = m - 2n + 1$。括号前是加号（因子 $+1$），各项符号原样不变。
  5. 操作去括号：$5 - (x - 4)$。
    括号前是减号，各项变号：$5 - x + 4$。（若合并常数：$= 9 - x$。）
  6. 操作去括号：$2(x + 3)$。
    分配律：$2 \times x + 2 \times 3 = 2x + 6$。
  7. 操作去括号：$-3(a - 2)$。
    $-3 \times a = -3a$，$-3 \times (-2) = +6$，结果 $-3a + 6$。
  8. 操作去括号并合并：$2(a + 1) - (a - 3)$。
    去括号：$2a + 2 - a + 3$（后一括号前是减号，$a$→$-a$、$-3$→$+3$）。合并：$(2a - a) + (2 + 3) = a + 5$。
  9. 操作化简：$3a + 2(a - 1)$。
    去括号 $3a + 2a - 2$，合并同类项 $3a + 2a = 5a$，结果 $5a - 2$。
  10. 反推去括号后得到 $x - y + 5$，原式是 $x + (\;\;)$ 还是 $x - (\;\;)$？括号里原来是什么？
    因为末两项 $-y+5$ 是变号后的结果，原式括号前应是减号：$x - (y - 5)$。（去括号：$y$→$-y$，$-5$→$+5$，正好得 $x - y + 5$。）
  11. 反推已知 $k(a - 2) = -2a + 4$，求括号前的数 $k$。
    第一项 $k \times a = -2a$，所以 $k = -2$。验证第二项：$-2 \times (-2) = +4$，与 $+4$ 一致，$k = -2$。
  12. 反推某式去括号后得 $3m - 6$，若括号里是 $(m - 2)$，括号前的数是几？
    $3m - 6 = 3 \times m + 3 \times (-2)$，所以括号前的数是 $3$，即 $3(m - 2)$。
  13. 辨错有人写 $-(x - 3) = -x - 3$。错在哪？正确结果是什么？
    错在只翻了第一项，第二项 $-3$ 忘了变号。括号前是减号，每一项都要翻号，$-3$ 应变 $+3$。正确 $-x + 3$。
  14. 辨错有人写 $-3(a - 2) = -3a - 6$。错在哪？
    第二项符号算错：$-3 \times (-2)$ 是负负得正 $= +6$，不是 $-6$。正确结果 $-3a + 6$。
  15. 辨错有人写 $5x - (2x - 3) = 5x - 2x - 3$。错在哪？
    括号前是减号，里面每一项都要变号，末项 $-3$ 应变 $+3$，被漏掉了。正确是 $5x - 2x + 3 = 3x + 3$。
